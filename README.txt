лабы и Рк в ветках
Ветка lab2 содержит решение 2 и 3 лабораторных
В ветке DZ 2 проекта, которые я делал во время семестра на Python.
  Первый это телеграм бот для изучения английского языка. В нем есть 2 режима игры: перевод слов и переевод картинок.
  В 1 режиме из БД выбирется слово которое игроку необхлодимо перевести, в случае успешного перевода пользователю начисляются очки рейтинга, в противном случае дается возможность использовать подскзку(выбор из 3 слов, 1 из которых правильное).
  Также после каждого раунда бот по запросу пользоваателя может отпрвить файл с произношением слова.
  Во 2 режиме пользовтелю предстоит перевести наа английский язык объект представленный на картинке. В остальном 2 режим схож с 1.
  Также реализованаа таблица лиеров по очкам рейтинга.

  Второй это культовая мобильная игра Doodle Jump. Выполнял ее для того чтобы ознакомится с pygame. 
  Есть авторизация, также топ, настройки. (Музыку можно выкключить в настройках). Управление наа стрелочки клавиатуры.
